# Stroke Prediction – Machine Learning Comparison

This project applies and compares multiple machine learning models for stroke prediction using a medical dataset. The models evaluated include Logistic Regression, Random Forest, and Gradient Boosting.

## Dataset

The dataset used in this project is publicly available on Kaggle:

https://www.kaggle.com/datasets/fedesoriano/stroke-prediction-dataset

## How to Run

1. Clone the repository:
git clone https://github.com/YOUR_USERNAME/stroke-ml-comparison.git

2. Install dependencies:
pip install -r requirements.txt

3. Download the dataset from Kaggle and place it in the project folder.

4. Run the notebook:
notebook.ipynb

## Project Structure

- notebook.ipynb → Full implementation (EDA, preprocessing, modeling, evaluation)
- requirements.txt → Required Python libraries
- README.md → Instructions

## Notes

- The project includes preprocessing, model training, hyperparameter tuning, and evaluation.
- Results can be reproduced by running the notebook from top to bottom.
